# 🎄 Aggressive Christmas Countdown 🎅

The **most intrusive, feature-rich, and festive** Christmas countdown website on the internet! This isn't your ordinary countdown timer - it's an immersive holiday experience that fills your screen with Christmas cheer... whether you want it or not! 😈

![Christmas Countdown Screenshot](https://github.com/user-attachments/assets/ab4ce422-e0c1-471c-84ae-bda0a556730f)

## ✨ Features

### 🎯 Core Features
- **Real-time Countdown**: Days, hours, minutes, and seconds until Christmas
- **Progress Bar**: Visual representation of the journey to Christmas
- **Animated Background**: Beautiful green gradient with festive vibes

### ❄️ Visual Effects
- **Snowfall Animation**: Realistic falling snow across the entire screen
- **Floating Ornaments**: Christmas decorations drifting around the page
- **Twinkling Lights**: Colorful Christmas lights at the top of the page
- **Particle Effects**: Confetti, fireworks, and snowstorm effects on demand
- **Glowing Animations**: Pulsing, floating, and shimmering effects everywhere

### 🎅 Interactive Components

#### Santa Tracker
- Real-time updates on Santa's location
- Gift preparation counter
- Progress bar showing preparation status

#### Christmas Facts
- 15+ interesting Christmas facts
- Auto-rotating facts every 15 seconds
- "New Fact" button for instant trivia

#### Wishlist Manager
- Add your Christmas wishes
- Persistent storage (saved in browser)
- Delete individual wishes
- Jingle sound on add

#### Naughty or Nice Meter
- Interactive morality meter
- "Be Nice" button to improve your score
- Score automatically fluctuates over time
- Color-coded rating system

#### Christmas Radio
- Synthesized Christmas music player
- Volume control
- Sound effects: Jingle bells, Ho Ho Ho, Bells
- Play/Stop controls

#### Christmas Messages
- Rotating festive messages
- 15+ different holiday greetings
- Auto-rotation every 10 seconds
- Manual refresh button

### 🎊 Intrusive Features (The "Aggressive" Part)

- **Popup Notifications**: Random Christmas messages appear every 30-60 seconds
- **Title Changes**: Page title rotates between festive messages every 3 seconds
- **Exit Warning**: Browser warns you before leaving (Christmas won't let you go!)
- **Random Screen Effects**: Occasional subtle effects to keep things interesting
- **Persistent Reminders**: Multiple notification sources for maximum holiday cheer

### 📊 Statistics Dashboard
- Visit counter (tracks how many times you've visited)
- Days since last Christmas
- Percentage of year completed
- Sleeps until Christmas

### ⚙️ Settings Panel
- Toggle snow effects
- Toggle popup notifications
- Toggle floating ornaments
- Toggle twinkling lights
- Auto-play music option

All settings are saved to browser localStorage!

## 🚀 Quick Start

### Option 1: Open Directly
Simply open `index.html` in your web browser!

### Option 2: Local Server
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server)
npx http-server

# PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.

## 🎨 Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Advanced animations, gradients, and responsive design
- **Vanilla JavaScript**: No dependencies, pure ES6+
- **Web Audio API**: Synthesized sounds and music
- **Canvas API**: Snow effect rendering
- **LocalStorage API**: Persistent user data

## 🎵 Audio Features

All sounds are synthesized using the Web Audio API - no audio files required! This includes:
- Background Christmas melody
- Jingle bell effects
- Ho Ho Ho sound
- Bell chimes

**Note**: Some browsers may require user interaction before playing audio.

## 🎁 Feature Highlights

### Particle System
Three types of particle effects:
- **Confetti Blast**: 100 colorful confetti pieces
- **Fireworks**: 5 bursts with 30 particles each
- **Snowstorm**: 200 snowflake emojis

### Animation System
Multiple CSS animations including:
- Pulse, float, and bounce effects
- Shimmer and glow effects
- Swing animations for ornaments
- Twinkle effects for lights
- Slide-in notifications

### Data Persistence
The following data is saved in your browser:
- Wishlist items
- Visit count
- Naughty/Nice score
- User preferences (settings)

## 🛠️ Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --christmas-red: #c41e3a;
    --christmas-green: #0f7d3e;
    --christmas-gold: #ffd700;
    /* ... more variables */
}
```

### Adjusting Effects
Modify parameters in `script.js`:
- Snowflake count
- Popup frequency
- Auto-rotation timers
- Particle counts
